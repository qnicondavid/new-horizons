# New-Horizons

---

**Technologies:** Oracle SQL Developer Data Modeler, Oracle SQL (DDL), Supabase (PostgreSQL), SQL  

Designed and implemented a complete relational database system for a travel agency, covering the full lifecycle of bookings and travel package management.

## Repository Structure

- **/diagrams** - logical and relational diagrams (Oracle SQL Developer Data Modeler)  
- **/sql** - Auto-generated DDL script, sample data population, and validation queries  
- **/report** - explanation of design choices  
- **/latex** - LaTeX source files of the technical report
